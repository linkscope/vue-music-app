# vue-music-app

模仿网易云音乐移动端页面，使用 Vue3.0 & TypeScript，为了更好地支持 TypeScript, 本项目采用 TSX 模板语法开发，仅供学习使用

本项目共分为五个大模块：推荐歌单、歌手列表、排行列表、搜索、音乐播放器(砍掉喜欢功能，起初准备对接微信/QQ 登录，后来觉得没多大必要，而且要做的更多，索性就做到这里，已经起到了学习的目的)

启动该项目(后台使用[网易云音乐接口](https://github.com/Binaryify/NeteaseCloudMusicApi)，已经运行在线上)

```
# step 1
yarn

# step 2
yarn serve
```
